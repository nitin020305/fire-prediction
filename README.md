Forest Fire Prediction (Flask + Random Forest)

A Machine Learning web application that predicts the probability of forest fire occurrence based on weather conditions.

The model uses a Random Forest Classifier trained on the Algerian Forest Fires dataset.

---

## 📌 Project Overview

This application predicts the probability of fire based on:

- Temperature
- Relative Humidity (RH)
- Wind Speed (Ws)
- Rain
- FFMC
- DMC
- ISI
- Region

The output includes:
- Fire Probability (%)
- Risk Level (High / Low)
